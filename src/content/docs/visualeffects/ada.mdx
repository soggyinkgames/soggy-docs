{/* vfx description */}
Inspired by Augusta Ada King, the first programer. Moving lines that cross over one another in two colors, in the shape of any 3d object. Visual Effects Graph has exposed variables to add a signed distance field file of 3d object and an exposed variable to edit line speed.
